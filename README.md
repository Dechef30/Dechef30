# Dechef30

Dechef30 的个人主页初始框架，围绕成长经历、兴趣、成绩成果、项目经历与联系方式展开。

## 本地预览

```powershell
python -m http.server 4173
```

然后打开 `http://127.0.0.1:4173/`。

## 页面结构

- 当前状态与个人简介
- 上海到广州的成长航线
- One Piece、Stephen Curry、BABYMONSTER 与 aespa 兴趣元素
- 成绩、荣誉、技能与项目占位结构
- 自定义熊掌光标、待机动态与响应式布局

## 后续内容

真实姓名、专业方向、成绩、获奖信息、项目资料和其他联系方式目前保留为待补充状态。
